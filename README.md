## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open `Settings` in GitHub.
4. Open `Pages`.
5. Set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. Save.

GitHub Pages will publish the website automatically.

## Notes

- Keep all files in the root exactly as they are now.
- Do not move `css`, `js`, or `assets` outside this structure.
- All links are already written for static hosting.
